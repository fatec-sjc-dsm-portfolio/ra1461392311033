Projeto API - Dados da Covid

Voltar para o Início

Objetivos

O objetivo deste projeto foi desenvolver uma API RESTful que permitisse a consulta de dados sobre a pandemia de Covid-19. A aplicação deveria fornecer estatísticas atualizadas, como número de casos, mortes e recuperados, que poderiam ser filtradas por país.

Tecnologias Utilizadas

  • Node.js
  • Express
  • MongoDB
  • Postman para testes

Resultados

O projeto foi bem-sucedido, permitindo a manipulação eficiente dos dados e a integração com outras aplicações. A API foi testada e validada utilizando o Postman, garantindo seu funcionamento adequado.
